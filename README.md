# UI-Vite 搭建官网
## 安装cva
``````
$ yarn global add create-vite-app
//或者
$ npm i -g create-vite-app
```
我这里用的是@1.18.0 创建项目目录
```
$ cva guli-ui
//或者
$ create-vite-app guli-ui //文件名自取
```
引入Vue Router 4 1、使用命令行查看vue-router的版本号
```
$ npm info vur-router versions
```
## 2、安装
